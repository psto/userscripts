## Installation

1. Install [Violentmonkey](https://github.com/violentmonkey/violentmonkey) extension for Firefox or Chrome
2. Open the extension's [Greasyfork](https://greasyfork.org) page
3. Click "Install this script"
4. Click "Install" in the Violentmonkey view
